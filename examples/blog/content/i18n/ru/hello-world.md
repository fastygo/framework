# Привет, мир

Это демонстрационный блог, построенный на **fastygo/framework** и **UI8Kit**.

Эта запись рендерится на сервере из markdown-файла, встроенного прямо
в бинарник. В формировании тела статьи не используется база данных, CMS
или клиентский JavaScript — markdown один раз конвертируется в HTML при старте
и кэшируется в памяти.

## Зачем предрендер?

- **Простая эксплуатация.** Один бинарник Go отдает и shell, и уже
  готовую HTML-страницу статьи.
- **Предсказуемая производительность.** Конвертация происходит на этапе запуска,
  а на запросы просто возвращается кэш.
- **Простое версионирование.** Посты лежат в git рядом с кодом.

## Что дальше?

Добавьте новый файл в `content/i18n/<locale>/<slug>.md`, добавьте соответствующую
запись в реестр постов в `internal/site/blog/feature.go` и пересоберите приложение.
